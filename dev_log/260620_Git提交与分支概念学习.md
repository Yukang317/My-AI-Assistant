## 讨论

**Q: 个人项目需要分分支吗？**

当前不需要。单分支（main）线性开发对 solo 项目足够。以下情况再开分支：
- 实验性功能（不确定是否会保留）
- 同时做两个互不相关的功能
- 修紧急 bug

## 学到的概念

- **原子提交**：一次提交只做一件事，方便以后回退和 review
- **暂存区控制**：`git add` 精确挑选文件，不一把梭 `git add .`
- **git 自动识别重命名**：按内容相似度判断，不靠文件名
