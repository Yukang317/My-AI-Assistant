# 7-0-b RAG 空结果降级（防幻觉）

**日期**：2026-06-27  
**状态**：待开始（7-0-a 验收后）  
**涉及文件**：`rag/rag_service.py`、`agent/tools/rag_search.py`、（可选）`agent/nodes/result_synthesis.py`

## 一句话目标

知识库检索零命中时，跳过 LLM 生成，明确返回「知识库中没有相关内容」，不让 Agent 编造答案。

## 背景问题

- 检索为空时 `_build_context` 只有占位提示，LLM 仍可能用自身知识编造
- `rag_search` 的 `if answer:` 无法区分「真查到」和「没查到但 LLM 瞎编」

## 你要做什么（7-0-a 完成后开始）

1. `rag_service.py`：恢复 `NO_RESULT_ANSWER` 常量 + `query()` / `query_stream()` 空结果短路
2. `rag_search.py`：识别 `no_result` 标记，返回 `success=True` + 诚实文案
3. （可选）`result_synthesis.py`：prompt 加 1 条「空检索不编造」规则

## 验收标准

- [ ] 问库外问题（如「火星移民政策」）→ 回复含「知识库中没有…」
- [ ] `rag_search` 返回 `success=True`（不是系统错误）
- [ ] 助手验收通过 → 删除对应 `_ref_` 参考文件

## 参考文件
