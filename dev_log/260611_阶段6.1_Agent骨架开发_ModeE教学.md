## 知识点记录

| 知识点 | 内容 |
|--------|------|
| TypedDict 函数内部定义 | 惰性求值保护，推迟类型注解求值到运行时，避免导入顺序导致 reducer 解析失败 |
| Annotated reducer | `list + operator.add` = 追加不覆盖；`list + add_messages` = 消息去重追加 |
| dataclass field(default_factory=dict) | Python 可变对象默认值陷阱——`artifacts: dict = {}` 所有实例共享同一个 dict |
| 不可变对象默认值安全 | `None`/`0`/`""`/`True` 不可原地修改，不需要 factory |
| ToolContext 桥梁角色 | 连接 Agent 编排层和业务数据层，通过 session_id 访问 PG 等外部存储 |
| ToolResult.error 用 str 不用 Exception | Exception 含不可序列化对象（traceback），LangGraph MemorySaver 序列化会失败 |
| typing.List vs 内置 list | `List[str]` 是类型注解，`list()` 是构造函数，不能混用 |
| 工具注册表 = 技能目录 | TOOLS dict 让 LLM 知道有什么工具可用，list_tools() 输出直接喂给 System Prompt |
