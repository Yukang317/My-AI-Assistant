
**为什么 RAG 要关？**  
前端路由：`useRag=true` → `/chat`（直调 RagService）；`useRag=false + 智能体 ON` → `/chat/graph`（Agent 全链路）。7-0-d 要测的是 **Agent 路由 → rag_search 工具 → 合成 → memory_update**，不是 RAG 直连接口。

> 7-0-c 验收时你用的是 RAG 开关 ON（直连接口），那次算「RAG 管道」预验证。7-0-d 用例 A 走 Agent 路径，是正式验收。
