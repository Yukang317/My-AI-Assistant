#### Step 10：`rag/evaluate.py` — RAG 评估模块（Mode E）

9 个函数，5 步评估管线：

```
① Milvus 随机采样父块 → LangChain Document
② RAGAS TestsetGenerator 自动出题（LLM-as-judge）
③ 调 RagService.query() 获取真实回答
④ RAGAS evaluate() 自动打分
⑤ 保存 JSON 报告 + 打印摘要
```

**核心设计决策**：
- **RAGAS LLM**：`ChatOpenAI` 指向 DeepSeek API（OpenAI 兼容）
- **RAGAS Embedding**：`HuggingFaceEmbeddings` 加载 BGE 本地模型（复用缓存，不额外占内存）
- **BM25 全量构建**：通过 `vs.client.query()` 取所有 child_chunks，IDF 依赖全局统计
- **NaN 安全**：`math.isnan`/`math.isinf` → None，保证 JSON 序列化

**Bug（2 个）**：328 行遗留代码（`list.items()` 崩溃）、459 行缩进错误

**其他踩坑**：pymilvus 未安装、网络超时（uv sync 大包）
